# Cara memasang mysql

- Buka folder : C:\xampp\mysql\bin
    ![1](../assets/img/mysql/1.png)
- Buka pengaturan environment
  1. ketik di pencarian : environment
  2. Environment Variables...
  3. Click Path
  4. Edit
  5. New
  6. paste 
- masuk cmd
- ketikkan :
    ```batch
        mysql -uroot -p
    ```
